To run the easy MDP
- run git apply diff_easyMDP
- follow instructions in README.md
- in Eclipse run EasyGridWorldLauncher.java

To run the easy MDP
- run git apply diff_hardMDP
- follow instructions in README.md
- in Eclipse run HardGridWorldLauncher.java

To run the the hard maze experiment
- run git apply diff_easyMDP
- follow instructions in README.md
  - run hardMaze.java instead of EasyGridWorldLauncher.java
